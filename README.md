# Earthly-Disasters
Interactive web map visualizing global natural disasters and hazard events using Mapbox GL JS and publicly available geospatial datasets.

Project Overview

Earthly Disasters is an interactive web mapping project that visualizes global natural hazard events using Mapbox GL JS and publicly available geospatial datasets. The project currently focuses on earthquake activity and will expand to include additional natural disasters such as volcanic eruptions, wildfires, hurricanes, and floods.

The goal of this project is to explore how interactive cartography and web GIS tools can communicate spatial patterns of natural hazards and improve public awareness of environmental risk.
